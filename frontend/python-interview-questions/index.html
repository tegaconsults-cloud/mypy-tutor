What is the difference between a list and a tuple?

Lists are mutable — you can change, add, or remove elements. Tuples are immutable — once created they cannot be modified. Tuples are faster and use less memory, making them ideal for fixed data.

What are decorators in Python?

Decorators are functions that wrap another function to extend its behaviour without modifying its source code. They use the @decorator_name syntax. Common examples: @staticmethod, @classmethod, @property.

Explain the difference between deep copy and shallow copy.

A shallow copy creates a new object but copies references to nested objects. A deep copy (via copy.deepcopy()) creates a completely independent clone, including all nested objects.

What is a generator and how does it differ from a list?

A generator uses yield to produce values lazily, one at a time, without storing them all in memory. This makes generators ideal for large datasets. A list stores all values upfront in memory.

What is the GIL and why does it matter?

The Global Interpreter Lock allows only one thread to execute Python bytecode at a time. This limits true multi-threading for CPU-bound tasks but doesn't affect I/O-bound tasks. Use multiprocessing for CPU parallelism.

What is the difference between __str__ and __repr__?

__str__ is called by print() and str() — intended for human-readable output. __repr__ is meant for developers and should ideally return a string that can reproduce the object.
